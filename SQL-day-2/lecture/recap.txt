Leçons clés de la journée

1. Requêtes SELECT les plus fréquentes en web dev:
  1.1 donne moi tout: SELECT * from xxxx
  1.2 donne moi les détails d'1 élément: SELECT * from xxx WHERE id = xxx
  1.3 filtres basiques: SELECT * from xxxx where city = 'Barcelona'
2. Syntaxe INSERT INTO
  2.1 SQL vous force de respecter la structure, mais s'en fiche du contenu
3. Syntaxe UPDATE & DELETE
  ATTENTION DE NE PAS OUBLIER LE WHERE!!!!!
4. Monde Ruby & DB:
  4.1 la DB choisit l'id, pas ruby
  4.2 les 2 mondes doivent toujours être synchronisés
  4.3 la méthode save doit être flexible!
  4.4 demandez-vous si une méthode doit plutôt être de classe ou d'instance
